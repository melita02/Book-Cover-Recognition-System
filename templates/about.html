About Us







Hey there, We created this web application which reads the text off of a book cover.

Image recognition, in the context of machine learning, is the ability of software to identify objects, places, people, writing and actions in images. In this project we applied Optical Character Recognition (OCR) to images using Tesseract and Python.

This application detects text of different font styles and sizes to help reading text off of a cover. For instance this can be applied in various applications such as on images uploaded on ecommerce websites to read text which makes random book recognition easier while shopping online.
